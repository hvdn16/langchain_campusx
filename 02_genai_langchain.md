## what is langchain 
its an open source framework that helps in building llm based applications 
it provides modular components and end to end tools that help developer build comprex ai apps such as chatbots question answering systems RAG, autonomous agents and more 
1. supports all major llms 
2. simplifies developing llm based applications 
3. integrations available for all major tools 
4. open source/free/actively developed 
5. supports all major genai use cases
